# Music Catalog

Веб-приложение для каталогизации и прослушивания музыки с возможностью регистрации пользователей.

## Функционал

- Просмотр каталога альбомов, музыкантов и треков
- Поиск по названию, жанру, году выпуска
- Встроенный аудиоплеер для воспроизведения треков
- Добавление треков в избранное
- Регистрация и авторизация пользователей
- Административная панель для управления контентом

## Запуск
`git clone <project name>`
`docker up --build `
